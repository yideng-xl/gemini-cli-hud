# Gemini CLI HUD Extension

## Overview
This extension provides a real-time, sticky status bar (HUD) at the bottom of your terminal during Gemini CLI sessions. It tracks token usage, context progress, and tool activity.

## Capability
- Sticky UI: Stays at the bottom of the terminal.
- Token Monitoring: Real-time display of token consumption.
- Context Tracking: Visual progress bar for context window usage.
- Tool Activity: Summary of tool calls (Bash, Read, Write, etc.).

## Usage
Automatically loads when the session starts.
